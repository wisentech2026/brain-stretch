## What it is

Brain Stretch is a small collection of attention-training exercises designed for one moment: **the urge to scroll**.

Instead of opening a short-video feed, you open this. You spend a minute on a focused exercise, and then you go back to what you were doing. The whole product is built around a single principle — **it should help you leave, not keep you here.**

换换脑子是一组对抗碎片化注意力的小练习。它存在的唯一目的，是在你想刷短视频的那一刻，提供一个安静的、有明确终点的替代选择——做完就走，回到该做的事。

---

## Design philosophy

This product is deliberately built *against* the patterns of attention-economy apps:

| Short-video mechanics | What we do instead |
|---|---|
| Endless scroll, no end | A clear timer; every session ends |
| Variable rewards | Quiet, deterministic feedback |
| Streaks & push notifications | None. Ever. |
| Optimized for time-on-app | Optimized for *leaving cleanly* |

If a feature would make someone unable to stop, it's the wrong feature — no matter how good the metrics look.

---

## The exercises

Each exercise is grounded in a recognized cognitive-science paradigm:

| Exercise | Paradigm | Trains |
|---|---|---|
| **Go / No-Go** | Go/No-Go · SART | Response inhibition |
| **Stroop** | Stroop Effect (1935) | Interference control |
| **Breathing** | 4-7-8 breathing | Arousal regulation |
| **Focused Attention** | FA meditation | Meta-awareness *(coming soon)* |
| **Visual Search** | Feature/conjunction search | Selective attention |
| **N-back** | N-back paradigm | Working memory |

## Tech

- Pure front-end. No backend, no account, no tracking.
- Progress is stored locally in the browser — never uploaded.
- Bilingual (中文 / English), auto-detected from system language.
- Procedurally synthesized audio (Web Audio API), no audio files.
- Built as a single self-contained HTML file.

---

## Privacy

Brain Stretch collects almost no data. Training records live only in your own browser's local storage. No login, no personal data on our servers. See [Privacy Policy](privacy.html).
